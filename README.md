A simple Blackjack game built with simple HTML, CSS and Javascript. 

How to Play
- Click Start Game to get your first card
- Click New Card to draw a new card
- Try getting exactly the sum of cards equal to 21.
    - If you are below the sum, draw a new card
    - If you are above the sum, the game is over and you need to restart

 Built With
 - HTML
 - CSS
 - Javascript
